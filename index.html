◎ 참여자 관리

기본 명단은 참여신청 구글폼 → 게시 CSV에서 자동으로 옵니다(새 학생이 폼을 내면 자동 추가). CSV 일괄 업로드는 설정 탭에서 할 수 있어요. 여기서는 폼을 못 낸 학생을 개별 추가합니다.

개별 추가한 학생은 이 기기에서만 명단에 보입니다. 그 학생이 일일인증 폼을 한 번이라도 제출하면 모든 기기에서 자동으로 나타나요. 가능하면 참여신청 폼을 내게 하는 게 가장 깔끔합니다.

✎ 잘못 입력한 정보 수정

학번을 틀리게 낸 학생은 학번 교정으로 고치세요. 그 학번으로 제출된 인증 기록까지 함께 옮겨집니다. 이름·별명만 고칠 수도 있어요.

⚙ 데이터 연결 (구글 시트 CSV)

구글폼 응답 시트를 웹에 CSV로 게시한 뒤 그 주소를 붙여넣으세요.

i지금 file://(로컬 파일)로 열고 있어요. 보안 정책상 온라인 CSV 자동 불러오기가 막힐 수 있습니다. 아래 CSV 파일 업로드를 쓰거나, GitHub Pages 등에 올려서 여세요.
구글 시트 CSV 게시 방법
  1. 구글폼 응답이 모이는 시트 열기 (폼 편집 → 응답 → 스프레드시트로 연결)
  2. 메뉴 파일 → 공유 → 웹에 게시
  3. ‘링크’ 탭에서 해당 시트 선택 → 형식 쉼표로 구분된 값(.csv)
  4. 게시 누르고 나온 주소를 위에 붙여넣기
한 번 설정하면 학생들이 폼을 낼 때마다 자동 반영됩니다(구글 게시 특성상 몇 분 지연될 수 있어요). 별명은 참여신청 명단 CSV가 연결돼야 순위표에 뜹니다.

또는 CSV 파일을 직접 올리기 — file:// 로컬 실행이나 자동 불러오기가 막힐 때 확실한 방법입니다. (시트에서 파일 → 다운로드 → 쉼표로 구분된 값(.csv))

파일 업로드 방식은 자동 새로고침이 없어요. 최신 순위를 보려면 시트를 다시 내려받아 올리면 됩니다.
